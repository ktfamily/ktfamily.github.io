Cách tạo web miễn phí với tên miền và hosting từ Github

Hiện để làm được website thì chúng ta cần có hosting (nơi lưu trữ website) và domain (tên miền). Đối với các b muốn vọc vạch để học tập hoặc chỉ đơn giản là làm trang giới thiệu chỉ với HTML, CSS thì nếu mua hosting và domain thì khá lãng phí, hiên github hỗ trợ chúng ta tạo những trang web tĩnh ngay trên này mà không cần bỏ thêm chi phí nào ?

Github là gì ?

GitHub là một dịch vụ cung cấp kho lưu trữ mã nguồn Git dựa trên nền web cho các dự án phát triển phần mềm. GitHub cung cấp cả phiên bản trả tiền lẫn miễn phí cho các tài khoản. Các dự án mã nguồn mở sẽ được cung cấp kho lưu trữ miễn phí.

Cách đăng ký tài khoản github

  1. Truy cập vào Github tại đây
  2. Nhấn vào sigup để bắt đầu đăng ký tài khoản
  3. Điền địa chỉ mail vào form
  4. Tạo pass
  5. Tạo username
  6. xác thực mail., hoàn tất b sẽ được chuyển đến dashboard
